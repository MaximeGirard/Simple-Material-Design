What is Simple Material Design ?

Simple Material Design is a library who contains some very basic and simple css rules to get Material Design elements in your website. They are realy easy to use. Moreover, you can modify the css finger in the nose ;).

How to use Simple Material Design ?

To use Simple Material Design, you just have to download the ressources files, include the include line in your html file and read this doc to know how to insert Material Design elements in your html page.
To include css and js:

<link href='path/to/simple-material-design.css' rel='stylesheet'>
<script src='path/to/simple-material-design.js'></script>

Documentation

Parameter

Some elements have parameters like background color, etc... To modify them just setup them in the :root selector in another stylesheet. Make sure to put this stylesheet after the Simple Material Design include.
Exemple :

:root {
--parameter: value;
}

Menu

To make a menu like this who is on his page, do:

<div class='smd-sidebar'>
<p><i class='smd-menu-button material-icons'>menu</i>
<span class='smd-menu-title'>Insert Title Here</span></p>
</div>

<div class='smd-menu-fond'>
</div>

<div class='smd-menu'>
<ul>
<li>Link 1</li>
<li>Link 2</li>
<li>Link 3</li>
<li>Link 4</li>
</ul>
</div>

You can modify these parameters :

--menubar-color: #212121; //color of the menu bar
--menubar-text-color: white //color of the menu bar text;

Input

Inputs look like that:



To get it, do:

<input type='text' placeholder='Input' class='smd-input' required>
//Change placeholder to change the label

You can modify this parameters :

--input-color: #2196F3; //Change the general color of the input

Button

There are three differents button.
The first one is the simple. It look like that :



To get it, do:

<button class='smd-simple-button'>Click Me !</button>

You can modify this parameters :

--simple-button-text-color: #03A9F4;



The second one is the important button. It look like that:



To get it, do:

<button class='smd-important-button'>Click Me !</button>

You can modify this parameters :

--important-button-color: rgb(33, 33, 33);
--important-button-text-color: #03A9F4;



The last one is the icon button. It look like that:



To change the icon, you need to change the test in the <i> tag.
Just read The Material Design Icon Documentation to see differents icons.

To get the button, do:

<button class='smd-icon-button'><i class='material-icons'>add</i></button>

You can modify these parameters :

--icon-button-color: rgb(33, 33, 33);
--icon-button-text-color: white;

Loaders

There are two differents loaders in this library.
The first one is a spinning loader:


To get it, do:

<div class='smd-loader-1'></div>

You can modify these parameters :

--loader-1-color: #3498db;
--loader-1-size: 120px;

The second one is a loading bar:



To get it, do:
<div class='smd-loader-2'><div class='smd-loader-2-bar'></div></div>

You can modify these parameters :

--loader-2-color: #3498db;
--loader-2-size: 120px;

Checkbox Inputs

Like classic checkbox but more cool ;)


To get it, do:

<label class="smd-checkbox-input-container">Checkbox Input
<input type="checkbox" name="checkbox">
<span class="smd-checkbox-input"></span>
</label>

You can modify these parameters :

--checkbox-input-color: #2196F3;

Radio Inputs

At the opposite of the checkbox, with the radio button, you can only select one choice.


To get it, do:

<label class="smd-radio-input-container">Radio Input
<input type="radio" name="radio">
<span class="smd-radio-input"></span>
</label>

You can modify these parameters :

--radio-input-color: #2196F3;
